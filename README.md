## ATLAS 项目概况

**ATLAS (Academic Task Learning Agent System)** 是一个基于 LangGraph 构建的复杂多智能体学术辅助系统，专为帮助学生管理学业而设计。

### 🎯 核心功能

1. **多智能体架构**
   - **PlannerAgent（规划智能体）**：负责日程安排、时间管理和学习计划生成
   - **NoteWriterAgent（笔记智能体）**：创建个性化的学习材料和内容摘要
   - **AdvisorAgent（顾问智能体）**：提供个性化的学术指导和支持

2. **智能协调机制**
   - 采用 ReACT（推理与行动）框架进行决策
   - 根据学生需求动态选择和协调不同的智能体
   - 支持并行执行多个智能体以提高效率

3. **个性化适配**
   - 适应学生的学习风格（视觉型、动觉型等）
   - 考虑特殊需求（如 ADHD 学生的注意力挑战）
   - 基于个人偏好定制支持方案

### 🏗️ 技术架构

- **工作流引擎**：使用 LangGraph 的状态图（StateGraph）进行编排
- **LLM 集成**：通过 NVIDIA NeMo API 调用大语言模型
- **异步处理**：全面支持 async/await 异步编程
- **跨平台支持**：兼容 Windows、Linux 和 macOS

### 💡 工作流程

1. **用户请求** → 输入学术相关的问题或需求
2. **协调器分析** → 分析请求复杂度，确定需要哪些智能体
3. **并行执行** → 同时运行多个兼容的智能体
4. **结果整合** → 汇总各智能体的输出，形成综合建议

### 🔧 主要特性

- ✅ 支持 ADHD 友好型学习模式
- ✅ 丰富的终端界面（使用 Rich 库）
- ✅ 虚拟环境自动配置
- ✅ 完善的错误处理和回退机制

### 📝 使用示例

用户可以提出各种学术问题，例如：
- "帮我准备下周的高级算法考试"
- "为机器学习概念创建学习笔记"
- "我需要帮助管理本月的多个截止日期"
- "考虑到我的 ADHD 和足球训练，规划我的学习时间表"
